| ![](/sites/default/files/isokm/km_link_gif.gif)**Дополнительно**   **Что такое Camel?** (Customised Applications for Mobile networks Enhanced Logic) — вид роуминга, который позволяет клиенту предоплаченной связи пользоваться за границей роумингом при любом положительном балансе не счету. CAMEL-роуминг предусматривает прямой набор номеров без использования дополнительных сервисных кодов.    **Что такое CallBack?** — телефонная услуга, позволяющая клиенту заказать соединение с нужным абонентом таким образом, чтобы звонок был входящим для них обоих. После совершения вызова клиент получит sms-сообщение с информацией о том, что его заявка принята. Через несколько секунд клиенту поступит звонок, на который ему следует ответить. После его ответа произойдет соединение с вызываемым абонентом.    **Как выбрать роуминг партнера?** Если подключение к сети зарубежного оператора не произошло автоматически, необходимо вручную в настройках телефона выбрать рекомендуемого оператора.    **Как подключить роуминг корпоративным клиентам?** Предоставить письмо на фирменном бланке организации, заверенное подписью уполномоченного лица и печатью организации.    **Работает ли раздача интернета в роуминге?** Да.    **ВАЖНО!** Не все роуминг-партнеры поддерживают услугу с устаревшей SIM-картой. Если у абонента SIM старого образца, необходимо произвести замену на новую. Новые SIM-карты содержат следующие начальные цифры идентификационных номеров:  |  |  | | --- | --- | | **Оператор** | **Идентификационный номер (новые SIM)** | | Tele2 | 899977771113  89997778002 | | ALTEL | 899970761113  899970771113  89997078003 |  Проверить поддерживает ли SIM клиента услугу роуминг можно на официальном сайте ([**Tele2**](https://www.tele2.kz/private/services/rouming/roaming) или **[ALTEL](https://www.altel.kz/uslugi/rouming)**) или в мобильном приложении | | | | | | |
