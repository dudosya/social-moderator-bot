| Активация абонентов **post-paid** производится специалистами ЦПиО **вручную** (в инвойсе)     Активация абонентов **pre-paid** производится в результате совершения абонентом голосового вызова **на номер **515** или посредством услуги передачи данных****.** После активации номера, абоненту будут высланы sms следующего содержания | | |
| **Наименование и параметры отправки SMS:** | | **Текст SMS:** |
| 1.      Приветствие (в течении 20 мин после активации номера в период с 9:00 – 21:00) | | *«Приветствуем Вас в сети АЛТЕЛ! Номер* *Call**С**entre**888»* |
| 2.      Выбор языка обслуживания (в течении 20 мин после сообщения «Приветствие») | | *«Для выбора языка информационного обслуживания наберите \*806#»* |
| **Дополнительная информация** | Услуга «Информирование новых абонентов после активации номера» предоставляет каждому абоненту  GSM сети возможность получать автоматические информационные SMS после активации абонентского номера  \* После 21:00 ч SMS  абоненту не отправляется. Сообщение будет доставлено утром.   \*\*Данная информация абонентам не предоставляется. | |
| Для активации номера необходимо выполнить следующие действия:  1. Отделить SIM-карту от пластиковой основы и установить ее в выключенное устройство. Необходимо сохранить пластиковую основу – она содержит PIN- и PUK-коды (под специальным стираемым слоем), необходимые для разблокировки SIM-карты.   2. Включить устройство, ввести код PIN11 и убедиться в том, что индикатор сигнала сети на телефоне стал активным.   3. Совершить голосовой вызов на номер 515 или воспользоваться услугой передачи данных | |
| **Примечания** | 1 Будьте внимательны, после 3-х неудачных попыток SIM-карта перейдет в режим повышенной безопасности и потребует ввода PUK1. После 10-ти неудачных попыток ввода PUK1 карта полностью блокируется и вы не сможете разблокировать ее самостоятельно.   2 Для активации посредством услуги передачи данных необходимо удостовериться, что в настройках мобильного устройства поле параметра APN (точка доступа) не является пустым. | |
